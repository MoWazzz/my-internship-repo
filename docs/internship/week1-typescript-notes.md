# what is typescript?
Typescript is JavaScript with types.

# what is a union?
A type that can be one of many types. For example, a variable that can be a string or a number.

# what is a Generic?
A reusable type using <T>.

# What is Async/Await?
 A way to write asynchronous code that looks synchronous. It is built on top of Promises.

 # why avoid any?
 Because it removes type safety and defeats the purpose of using TypeScript. It is better to use unknown or a specific type.
